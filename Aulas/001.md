# Dicas do terminal

Vou listar alguns comandos que estamos aprendendo:
- `cd` (navega entre pastas)

Exemplo para entrar nan pasta:
```
cd nomaDaPasta
```

Exemplo para sair de uma pasta:
```
cd ..
```

- `mkdir` (cria pastas)
Exeplo de com criar uma pasta:
```
mkdir nomaDaPasta
```
